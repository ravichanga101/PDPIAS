Web Technologies


Web Technologies Lab of IT department resides at lab no. 404-B 1st floor of CE/IT building.It is primarily used for the courses related to web technologies. It contains 45 latest Dell Optiplex machines and also provides suitable space for students to work together and interact with their instructor. The machines are equipped with relevant software tools like PHP, Web server, Wamp, Macromedia Dream Weaver and Net Beans.


Students work with Markup Languages like HTTP, scripting languages like PHP as well as JAVA and use utilities for designing website graphics like Flash. The lab also has adequate projection facilities for presentations so that both the instructor can deliver the lecture content and the students can present their Projects : effectively.
